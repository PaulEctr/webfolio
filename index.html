Pourquoi ce projet ?

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the style sheet into this very page. The HTML remains the same, the only thing that has changed is the external CSS file. Yes, really.

CSS allows complete and total control over the style of a hypertext document. The only way this can be illustrated in a way that gets people excited is by demonstrating what it can truly be, once the reins are placed in the hands of those able to create beauty from structure. Designers and coders alike have contributed to the beauty of the web; we can always push it further.

Qu'est ce qu'UPIAN

Upian est une société dondé en 1998 par Alexandre Brachet.

Cette société est impliqué dans pleins de dommaines différents, tous touchant parfois à la programation, et pour être même plus précis, à la progrmmation web. Ils font également de la production de contenu, sur Snapchat pour Arte par exemple, à travers le compte ARTE FAQ. Mais aussi sur instagram pour Dargaud avec "Matin quel journal !".

Pour ce qui est de la programation, ils font de nombreuses préstations pour de nombreux clients, ce qui peux passer de la réalisation du questionnaire en ligne "Il était temps" pour Arte, à la création d'un jeu vidéo, j'ai nommé Unmaze, en passant bien evidement par de la création de site internet.

Requirements

Where possible, we would like to see mostly CSS 1 & 2 usage. CSS 3 & 4 should be limited to widely-supported elements only, or strong fallbacks should be provided. The CSS Zen Garden is about functional, practical CSS and not the latest bleeding-edge tricks viewable by 2% of the browsing public. The only real requirement we have is that your CSS validates.

Luckily, designing this way shows how well various browsers have implemented CSS by now. When sticking to the guidelines you should see fairly consistent results across most modern browsers. Due to the sheer number of user agents on the web these days — especially when you factor in mobile — pixel-perfect layouts may not be possible across every platform. That’s okay, but do test in as many as you can. Your design should work in at least IE9+ and the latest Chrome, Firefox, iOS and Android browsers (run by over 90% of the population).

We ask that you submit original artwork. Please respect copyright laws. Please keep objectionable material to a minimum, and try to incorporate unique and interesting visual themes to your work. We’re well past the point of needing another garden-related design.

This is a learning exercise as well as a demonstration. You retain full copyright on your graphics (with limited exceptions, see submission guidelines), but we ask you release your CSS under a Creative Commons license identical to the one on this site so that others may learn from your work.
